# Martial Arts Curriculum Builder

Two modes. **Curriculum mode** builds a multi-week training plan from your grading syllabus (great for "what should each belt/grade train between now and the next test"). **Lesson mode** builds one class. Both end with a promo caption so every plan doubles as marketing.

## When to use
- Mapping a month (or a grading cycle) of classes for a level
- Planning a single class from one technique
- Turning the curriculum you carry in your head into a documented, sellable asset

## Inputs (ask once if missing, then proceed with sensible defaults)
- **Mode:** curriculum or lesson
- **Style:** BJJ / Aikido / Muay Thai / MMA (default BJJ)
- **Level / grade:** e.g. white belt, blue belt, 5th Kyu, kids
- **Syllabus or technique list:** paste the grading techniques if you have them; if not, the skill proposes a standard set for that level and flags it as "confirm against your syllabus"
- Curriculum mode also needs: **training days per week** and **time horizon** (e.g. 4 weeks, or "until next grading")
- Lesson mode also needs: **technique/theme** and **class length in minutes**

## Steps
### Curriculum mode
1. Confirm inputs.
2. Output a **week-by-week plan** for the horizon. For each week:
   - **Focus** of the week (one theme)
   - **Techniques** to cover, drawn from the syllabus
   - **Per training day** split, so techniques are distributed (not crammed)
   - **Review/spaced repetition** note: which prior-week techniques to revisit
3. Add a **Grading-readiness check**: the few techniques most often missed before a test for this level.

### Lesson mode
1. Confirm inputs.
2. Output a single **Lesson Plan**: Objective, Warm-up (min), Technical breakdown (numbered, each with a cue + a common mistake), Drilling progression (static → resistance → live, minutes per stage summing to class length), 3 coaching cues, Homework.

### Both modes finish with
3. **Promo caption** (under 60 words) for Instagram/email: benefit, not hype, with a soft invite to a trial class.
4. **Packaging note**: one line on how this becomes a sellable asset (online curriculum, PDF, paid seminar, members area).

## Output rules
- In lesson mode, the minutes must add up to the class length.
- Plain language a brand-new coach could teach from cold.
- If a syllabus wasn't provided, clearly mark proposed techniques as "confirm against your school's syllabus."
